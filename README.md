# goowe-python
Implementation of GOOWE (Bonab and Can, 2017) in Python using scikit-multiflow

### Status
Working prototype of GOOWE (single-label version). Run experiment.py for a simple run that demonstrates the functionality.

### Requirements
* numpy
* scikit-multiflow
